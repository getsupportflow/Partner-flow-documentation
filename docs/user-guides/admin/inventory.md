# Inventory Management — Admin

The Admin Inventory section is designed for service creators who manage stock across services and providers. This area focuses on defining and controlling the product catalogue and provides insights into day-to-day usage to optimise operations and planning.

## Key Functions

### Upload Stock Information

Add new products with essential details:

- **SKU** and **EAN / barcode numbers**
- **Product name and description**
- **Category** for organisation
- **Compatible models or services** (optional, to ensure correct allocation)

### Add Product Images

Attach images to each product so providers can quickly identify the correct item at a glance.

### Set Advisory Thresholds

Define minimum stock levels for each product. When a provider's stock drops below the threshold, the system can:

- Alert the relevant parties
- Automatically draft a purchase order for replenishment

### Bulk Import and Updates

Use CSV import or integration tools to upload or update multiple products at once. This is particularly useful when setting up a large catalogue or syncing with an existing inventory system.

### Usage Reporting

Generate reports showing:

- Which parts are used most frequently
- Which service providers are using which parts
- Usage trends over time by service or region

These insights help with stock planning, purchasing decisions, and service optimisation.

## Benefits

| Benefit | Detail |
|---|---|
| **Central control** | Maintain a master record of all approved stock items across services |
| **Clarity** | Product images and model compatibility make it easy to identify the right items |
| **Insightful analytics** | Track usage patterns to optimise inventory levels and anticipate demand |
| **Accuracy** | Providers only use approved items with clear compatibility guidelines |
| **Efficiency** | Predefined thresholds and product details automate replenishment and reduce errors |
| **Scalability** | Quickly add or update items to support new services, models, or industries |
